Medicare+ – Medical Centre Management System
Project Overview

The desktop application (internal system) was developed for Dr. Athula Chandrasiri Medical Centre staff, including doctor, assistants, pharmacist, lab staff, and admin.
It provides a centralized and secure way to manage patients, prescriptions, pharmacy, staff, laboratory, and finances.

 Features (Console/Desktop App)
Doctor & Assistants

Enter diagnosis history & patient details

Send prescriptions digitally to pharmacist

Access patient medical records

Generate medical certificates

 Pharmacist

Receive prescriptions digitally

Manage drug inventory (stock, expiry, supplier details)

Update drug availability in real-time

 Lab Assistant

Receive test requests directly from the doctor

Upload and release lab reports to the system

Notify patients & doctor of results

 Admin / Doctor (Owner)

Manage staff & employee registration

Generate financial reports (income statements, progress tracking)

Maintain billing history & invoices

🛠 Tech Stack

Frontend: C#, Windows Forms

Database: Microsoft SQL Server

Tools: Git, Visual Studio, SSMS

 Future Enhancements
